# 📺 Fixed-Screen

**Fixed-Screen** é um aplicativo desktop desenvolvido com **Tauri + React + TypeScript**, que permite assistir vídeos ou transmissões ao vivo em janelas flutuantes, sempre visíveis na tela, enquanto você realiza outras tarefas.  
O objetivo é facilitar a multitarefa, permitindo abrir **um ou múltiplos vídeos flutuantes** sem precisar alternar de janela constantemente.

---

## 🚀 Funcionalidades

- Assista vídeos em **janelas flutuantes fixas** (sempre no topo).
- Suporte a **múltiplos vídeos** simultaneamente.
- Interface simples e intuitiva feita em **React + TailwindCSS**.
- Aplicativo leve e rápido graças ao **Tauri**.
- Compatível com Windows (outros sistemas em breve).

---

## 🛠️ Tecnologias Utilizadas

- [Tauri](https://tauri.app/) – Aplicativos desktop leves
- [React](https://react.dev/) – Interface de usuário
- [TypeScript](https://www.typescriptlang.org/) – Tipagem estática
- [Vite](https://vitejs.dev/) – Build e dev server

## 💻 Instalação e Uso

1. Baixe o instalador acima.
2. Execute o arquivo `fixed-screen_1.0.0_x64-setup.exe`.
3. Abra o aplicativo e adicione o link do vídeo ou live que deseja assistir.
4. Aproveite assistir seus vídeos em janelas flutuantes enquanto trabalha!
